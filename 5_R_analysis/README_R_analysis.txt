### README 

- Scripts for R analysis :

    * raw.v.filtered.R     : parsing multiqc data
    * read_depth.R         : get per sample number of reads
    * data_analysis.R      : master script that runs DEseq2, plots PCA, plots correlation plot, and differential strip plots
    * gsea.R               : Runs GSEA on human mapped GO terms using DEseq2 results
    * GO_bubble_plots_v2.R : Plot bubble plot result of GO enrichment analysis
